# ardour-qml

QML bindings for Ardour lib.

## Required packages

At least Qt 6.9 is required.

Debian:

```
vamp-plugin-sdk libarchive-dev lv2-dev libfftw3-dev liblilv-dev liblo-dev librubberband-dev \
libsoundtouch-dev libltc-dev libsndfile1-dev libtag1-dev libsigc++-2.0-dev libboost-dev \
libcurl4-openssl-dev libglib2.0-dev libglibmm-2.4-dev qt6-shadertools-dev qt6-base-private-dev
```

Homebrew:
```
glibmm liblo lilv fftw lv2 libarchive libsamplerate vamp-plugin-sdk boost rubberband sound-touch libltc taglib
```
